# Telegram Bot DDoS

텔레그램 봇 DDoS는 텔레그램 Bot 시스템을 사용해서 제작된 DDoS 도구입니다.

텔레그램 봇 처럼 작동하며, 공격자는 봇을 등록하고 Telegram Id, Chat Id만 작성 후 라이브러리 링크를 맞게끔 연결하면 작동될 겁니다.

텔레그램 봇 채팅으로 공격을 지시할 수 있으며, 여러가지 기능을 제공합니다.

현재는 개발 중단되어 더 이상 관리되지 않습니다.

필요한 라이브러리: boost OpenSSL ws2_32.lib gdi32.lib advapi32.lib crypt32.lib user32.lib tgbot.lib

## 중요: 윤리적 사용 및 법적 고지 (Ethical Use & Disclaimer)

**이 도구는 오로지 교육적인 목적과 승인된 환경에서의 보안 테스트를 위해서만 제작되었습니다.**

저자는 이 도구를 불법적인 활동에 사용하는 것을 강력히 반대합니다. 승인되지 않은 대상(웹사이트, 서버, IP 등)에 대해 이 도구를 사용하는 것은 명백한 **불법**이며, 심각한 법적 처벌을 받을 수 있습니다.

**자신이 소유하거나, 명시적인 서면 허가를 받은 시스템에 대해서만 사용하십시오.** 이 도구의 사용으로 발생하는 모든 법적 책임은 전적으로 사용자에게 있습니다.

### 프로젝트 목적 (Purpose)

이 프로젝트는 네트워크 트래픽 생성의 원리를 학습하고, 다음과 같은 DDoS 공격 기법을 **방어하는 방법**을 연구하기 위해 시작되었습니다.

이 코드를 분석함으로써 개발자는 네트워크 프로토콜의 취약점을 이해하고, **더 나은 방어 시스템** (예: 방화벽 룰셋, 트래픽 필터링 로직)을 구축하는 데 도움을 받을 수 있습니다.

**절대로 외부 공용 IP에 테스트하지 마십시오.**
